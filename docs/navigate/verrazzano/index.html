Verrazzano

Verrazzanoセクションとその4つのサブセクションは、WebLogicドメインがVerrazzanoによって管理されるKubernetesクラスタへのWebLogicベースのアプリケーションのデプロイをサポートします。 役立つセクションがあります:

  1. Kubernetesクライアント(kubectl)を「構成」して、Kubernetesクラスタに接続します。
  2. 「インストール」 Verrazzano.
  3. WebLogicドメインのVerrazzanoコンポーネントを「作成とデプロイ」します。
  4. Verrazzanoへのアプリケーションの「デプロイ」。

    クライアント構成

    クライアント構成では、Kubernetesクラスタへの必要な接続を取得できます。

    インストール

    VerrazzanoをターゲットKubernetesクラスタにインストールします。

    コンポーネント

    WebLogicドメインのVerrazzanoコンポーネントを作成してデプロイします。

    アプリケーション

    Verrazzanoアプリケーションを定義およびデプロイします。
