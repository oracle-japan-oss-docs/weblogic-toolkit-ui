機械翻訳について

Kubernetes

Kubernetesセクションとその4つのサブセクションは、WebLogicドメインがWebLogic Kubernetes Operatorで管理されるKubernetesクラスタへのWebLogicベースのアプリケーションのデプロイをサポートします。 役立つセクションがあります:

  1. Kubernetesクライアント(kubectl)を「構成」して、Kubernetesクラスタに接続します。
  2. WebLogic Kubernetes Operatorを「インストール」します。
  3. WebLogic Kubernetes Operatorで使用されるWebLogicドメインのDomainリソース構成を「デプロイ」します。
  4. 必要に応じてイングレス・コントローラを「インストール」し、Kubernetesクラスタの外部からWebLogicドメインへのアクセスを許可するイングレス・ルートを追加します。

    クライアント構成

    クライアント構成では、Kubernetesクラスタへの必要な接続を取得できます。

    WebLogic Kubernetes Operator

    ターゲットKubernetesクラスタにWebLogic Kubernetes Operatorをインストールします。

    WebLogicドメイン

    WebLogicドメインのKubernetesカスタム・リソースを作成してデプロイします。

    イングレス・コントローラ

    イングレス・コントローラをインストールして構成します。
