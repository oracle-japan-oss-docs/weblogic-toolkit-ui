ToDo Listアプリケーションへのアクセス

アプリケーションがデプロイされ、WebLogicドメインが起動して実行されたら、「URLの更新」をクリックして、次のイメージに示すように、URLを実際のホスト名で更新すると、リンクもアクティブ化されます。

次の図に示すように、リンクをクリックしてアプリケーションに移動します。 インストールで自己署名証明書が使用されているため、アプリケーションへの移動が安全でないことをブラウザから警告される可能性があります。 これは、環境の例では一般的です。 これで、ToDoリスト・アプリケーションを使用できます。
