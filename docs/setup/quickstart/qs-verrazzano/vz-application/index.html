Verrazzanoアプリケーションのデプロイ

Verrazzanoアプリケーションは、それぞれゼロ個以上の特性がアタッチされたコンポーネントのリストです。 Verrazzano > Applicationページに移動します。 次の図に示すように、5つのコンポーネントをすべて追加し、残りのフィールドに入力します。

ここで、todolist-domainコンポーネントを展開し、Enable Ingress Traitを有効にして、Ingress Rules表のプラス記号をクリックします。 次の図に示すデータを、結果のフォームに入力します。

「アプリケーションのデプロイ」をクリックします。 アクションが完了した後も、アプリケーションが完全にデプロイされるまで数分かかります。 「アプリケーション・ステータスの取得」を使用して、進行状況を確認します。
