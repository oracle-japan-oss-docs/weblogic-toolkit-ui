WebLogic Kubernetes Toolkit UIドキュメント

WebLogic Kubernetes Toolkit (WKT)は、KubernetesクラスタのLinuxコンテナで実行するためにWebLogicベースのアプリケーションをプロビジョニングするのに役立つオープン・ソース・ツールのコレクションです。 WKTには、次のツールが含まれています:

  • WebLogic Deploy Tooling (WDT) - WebLogicドメインの単一のメタデータ・モデル表現から動作する、単一目的ライフサイクル・ツールのセット。
  • WebLogic Image Tool (WIT) - WebLogicドメインを実行するためのLinuxコンテナ・イメージを作成するためのツール。
  • WebLogic Kubernetes Operator (WKO) - KubernetesクラスタでWebLogicドメインをネイティブに実行できるようにするKubernetesオペレータ。
  • WebLogic Remote Console - どこで実行されているWebLogic Serverドメインを管理するための軽量コンソール。

WKT UIは、WKTツール、Docker、HelmおよびKubernetesクライアント(kubectl)をラップするグラフィカル・ユーザー・インタフェースを提供し、作成および変更プロセスをガイドします。WebLogicドメインのモデル、ドメインの実行に使用するLinuxコンテナ・イメージの作成、およびKubernetesクラスタ内のドメインのデプロイとアクセスに必要なソフトウェアおよび構成の設定とデプロイ。

チュートリアルについては、「クイック・スタート」ガイドを確認してください。このガイドでは、WebLogic Kubernetes Toolkit UI (WKTUI)の機能について説明し、その機能を理解します。
